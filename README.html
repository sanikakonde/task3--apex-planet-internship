UserMgmt Pro - Task 3 Project

Features

  • Authentication (Register/Login/Logout) with Sessions
  • Password hashing using password_hash()
  • Role Based Login (Admin/User)
  • CRUD operations for Users (Admin Panel)
  • Delete confirmation popup modal
  • Prepared Statements to prevent SQL Injection
  • Server-side input validation
  • Profile Management with Profile Picture Upload + size/type validation
  • Dashboard Analytics (Total Users/Admin count)

How to Run (XAMPP)

  1. Extract project folder into C:/xampp/htdocs/
  2. Start Apache + MySQL from XAMPP Control Panel
  3. Open phpMyAdmin
  4. Create database: user_mgmt_pro
  5. Import SQL file: sql/user_mgmt_pro.sql
  6. Run project: http://localhost/user_mgmt_pro/
